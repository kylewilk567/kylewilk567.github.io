FDM 3D Printing Fault Detection

I worked with the Mechanical Engineering Department at UM to develop software for a low-cost, consumer-grade touch probe sensor that can accurately detect a wide-range of faults. The software was developed as a plugin to Octoprint, a popular 3D printing application that allows the user to control their printer remotely. You can find more about the project in this published article.

FDM 3D Printing Fault Prediction

The goal of this project was to create a neural network that can accurately predict if a fault will occur given certain printer settings and a GCODE file. Neural networks require a large amount of data to train; the touch probe used in the Fault Detection project (see on the left) was used to gather this data. The neural network created could predict layer shifting with accuracy above 99%.

RoadBuilder (Pathfinding API)

A Java pathfinding plugin for Minecraft that traverses a 3D "blocky" world. Supports different path-finding and path-building algorithms at runtime (strategy design pattern). Stores data in MySQL database. Uses a greedy algorithm to generate paths quickly. Project in-progress (estimated completion Dec. 22).

3D Printing Fault Detection Probe (1/3)

Touch-probe using hall effect sensor and push-pull solenoid detects a wide-range faults in FDM 3D printers. I used a Python multiprocessing library to generate sample points while printing allowing for more computation time with minimal effect to print time. Sample points are also generated proportionally to the cross-sectional area of the 3D print to improve detection on larger layers. This is done using density clustering to divide the layer into regions and then summing the area of each cluster's convex hull.

3D Printing Fault Detection Probe (2/3)

I created a user interface including a sample-point viewer with pan and zoom controls, layer selection buttons, and sample-point settings.

3D Printing Fault Detection Probe (3/3)

I used Solidworks to modify a bracket to attach the probe to a Prusa MK3S printer. The bracket is easily attached using two screws through the threaded inserts of the fan.

Eagle Scout Project

I built three Little Free Pantries for elementary schools in the Walled Lake Consolidated School District. The project initially was to build one pantry. After learning about a classmate who I knew for several years and was in need of food, I decided to build more to help families like hers. The reason for Little Free Pantries is more than just giving food to the community. It's a way for people in need to take without having to ask and teaches younger students about food poverty.

The project was a test to my leadership ability and flexibility. I worked with my troop to build the pantries and planned ahead by creating stations and diagrams to help communicate the jobs that needed to be done.
