About me

I am a junior studying Computer Science and Electrical Engineering at the University of Michigan. My interests are in efficient, scalable algorithmic programming and machine learning. I am searching for a company with a hands-on, collaborative environment related to software development and/or robotics.

FDM 3D Printing Fault Detection

I worked with the Mechanical Engineering Department at UM to develop software for a low-cost, consumer-grade touch probe sensor that can accurately detect a wide-range of faults. The software was developed as a plugin to Octoprint, a popular 3D printing application that allows the user to control their printer remotely. You can find more about the project on the Gitlab Repo here

FDM 3D Printing Fault Prediction

The goal of this project is to create a neural network that can accurately predict if a fault will occur given certain printer settings and a GCODE file. Neural networks require a large amount of data to train; The touch probe used in the Fault Detection project (see on the left) will be used to gather this data.

Game Development

Projects in this space are my hobby. I create plugins for Minecraft for fun. Some of my projects include the following:

  • SupportBot - Utilizes a neural network to respond to player's messages so staff don't have to. Messages are parsed using a bag-of-words model and input into the neural network to give 1 of 24 responses (or no response)
  • Chest Regeneration - Allows chests to regenerate after a configurable amount of time and for admin to create custom loot-tables for each chest-type. Chest data is stored in a MySQL database.
